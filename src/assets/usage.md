# 安装

```bash
npm i ng-marked-preview
```
# 说明

> 当前最新版本：0.0.2，在线尝试示例[]()

# 引入样式

在`angular.json`中引入

```json
{
  "styles": [
    "node_modules/ng-marked-preview/style/theme/default.less"
  ]
}
```

在`style.less`中引入

```less
@import "~ng-marked-preview/style/theme/default.less";
```


在`style.css`中引入
```css
@import "~ng-marked-preview/style/theme/default.css";
```
